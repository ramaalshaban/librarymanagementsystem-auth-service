Available Routes

  • registerUser => POST /registeruser
  • updateUser => PATCH /users/:userId
  • deleteUser => DELETE /users/:userId
  • updateUserRole => PATCH /userrole/:userId
  • updatePassword => PATCH /password/:userId
  • getUser => GET /users/:userId
  • listUsers => GET /users
  • createGroup => POST /group
  • updateGroup => PATCH /group/:userGroupId
  • getGroup => GET /group/:userGroupId
  • listGroups => GET /groups
  • createGroupMember => POST /groupmember
  • deleteGroupMember => DELETE /groupmember/:userGroupMemberId
  • getGroupMember => GET /groupmember/:userGroupMemberId
  • listGroupMembers => GET /groupmembers
